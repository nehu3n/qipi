Qipi is a universal package manager. It focuses on efficiency and unification of packages from different languages and environments in a single tool, seeking to be as fast and simple as possible.

## ✨ Features:

- 🚀 **Fast:** Using concurrency, packet fetch, read and write operations are done as fast as possible.

- 💿 **Efficient:** All packages are stored in a single system folder, avoiding redundancy in each project through symlinks.

- 🌍 **Universal:** Manager available for JavaScript/TypeScript, Python and Ruby. _More coming soon._

- 🧩 **Simple:** An intuitive terminal cli and interface makes things easier.

## 📓 Documentation
